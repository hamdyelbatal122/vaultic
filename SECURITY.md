## Reporting a Vulnerability

Please do not open public issues for vulnerabilities.

Report security issues privately by opening a confidential security advisory in GitHub Security tab for this repository.

Include:

1. A clear description of the issue.
2. Steps to reproduce.
3. Affected Vaultic tag and Laravel/PHP versions.
4. Suggested remediation if available.

We will acknowledge the report and provide status updates as investigation progresses.
